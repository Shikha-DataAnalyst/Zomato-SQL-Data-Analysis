# Zomato-SQL-Data-Analysis

## Project Overview:
This project analyzes Zomato restaurant data using SQL to identify customer trends, restaurant performance, cuisine popularity, and delivery behavior.

## Skills Demonstrated:
- SQL Joins
- Aggregations
- GROUP BY
- Subqueries
- Case When
- Business Insights
  
## Business Problems Solved:
- Top rated restaurants
- Indian Restaurant Market statistics
- top most popular cuisine types and their performance metrics in India
- Online delivery impact
- City-wise restaurant distribution
- Restaurant voting analysis
- Market Penetration Analysis - India vs International

## Key Insights:
- Online delivery restaurants showed better customer engagement.
- North Indian cuisine dominated restaurant listings.
- High-rated restaurants received significantly higher votes.

## Dataset:
Zomato restaurant dataset containing restaurant details, cuisines, ratings, pricing, and delivery information.
